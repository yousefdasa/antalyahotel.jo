# Antalya Hotel Amman - Deployment Guide

هذا الدليل سيساعدك على جعل موقع فندق أنطاليا "لايف" (مباشر) وتحديثه بسهولة.

## 1. الخطوات الأولى (GitHub)
1. قم بإنشاء حساب على [GitHub](https://github.com).
2. أنشئ مستودعاً جديداً (New Repository) باسم `antalya-hotel`.
3. ارفع كافة ملفات المشروع إلى هذا المستودع.

## 2. الربط بمنصة النشر (Vercel)
1. ادخل إلى [Vercel](https://vercel.com) وسجل دخولك باستخدام حساب GitHub.
2. اضغط على **"Add New"** ثم **"Project"**.
3. اختر مستودع `antalya-hotel` من القائمة.
4. **هام جداً (Environment Variables):** قبل الضغط على Deploy، ابحث عن قسم "Environment Variables" وأضف المفتاح التالي:
   - **Key:** `API_KEY`
   - **Value:** (ضع مفتاح Gemini API الخاص بك هنا)
5. اضغط على **"Deploy"**. مبروك! موقعك الآن مباشر.

## 3. كيف تقوم بالتحديث عن طريق المساعد الذكي؟
عندما أطلب منك إجراء تعديلات:
1. انسخ الكود الجديد الذي سأعطيك إياه.
2. اذهب إلى ملفك في GitHub (مثلاً `App.tsx`).
3. اضغط على أيقونة القلم (Edit).
4. استبدل الكود القديم بالجديد واضغط **"Commit changes"**.
5. سيقوم Vercel بالاستشعار تلقائياً وبدء تحديث الموقع Live خلال ثوانٍ.

## 4. نصيحة الأمان
- لا تضع مفتاح الـ API داخل الكود أبداً. دائماً استخدم إعدادات Vercel (Environment Variables) كما هو موضح أعلاه.
